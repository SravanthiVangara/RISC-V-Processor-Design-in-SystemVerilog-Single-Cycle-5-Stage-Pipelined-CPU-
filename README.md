# RISC-V-Processor-Design-in-SystemVerilog-Single-Cycle-5-Stage-Pipelined-CPU-
Design and verification of a RISC-V processor in SystemVerilog, starting from fundamental components (ALU, Register File, Memories,etc) to a fully functional 5-stage pipelined CPU with hazard detection, forwarding unit, assertions, and structured testbench environment.
